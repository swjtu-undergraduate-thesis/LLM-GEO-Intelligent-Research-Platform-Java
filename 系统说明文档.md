### 一、系统概览

LLM-GEO智能研究平台包含：
1.基于 Spring Boot（RuoYi） 的后端，位于 `dtm_java`（java环境配置、maven配置、redis配置）
2.基于 Vue 2 + Element UI 的前端，位于 `dtm_vue`（nodejs配置）

### 二、项目架构

#### 1.Java后端

1.1 Maven分模块构建

ruoyi-admin：后端主工程与启动入口，包含控制器、鉴权入口、Spring Boot 启动类、全局配置，开发时在controller目录下设计控制层，在下图中controller路径下创建各自模块的目录并存放controller代码

![image-20260311021914493](C:\Users\zjq\AppData\Roaming\Typora\typora-user-images\image-20260311021914493.png)

ruoyi-system：系统业务模块（用户、角色、菜单、字典、日志等核心业务功能），开发时在`src/main/java/com/ruoyi`下创建自己对应的业务模块，在业务模块下创建`domain、service、mapper`文件夹分别用于存储实体类、服务层代码、映射层接口代码，同时在`src/resources/mapper`下创建对应main中的模块文件夹，用于存储对应映射层接口的实现代码

![image-20260311022155367](C:\Users\zjq\AppData\Roaming\Typora\typora-user-images\image-20260311022155367.png)

ruoyi-common：通用工具与基础组件（通用工具类、常量、通用响应、HTTP 客户端等），包含 FlaskApiClient 集成 Flask 分析服务

ruoyi-quartz：定时任务调度模块（Quartz 任务定义与执行）

ruoyi-generator：代码生成模块（基于数据库表结构生成 CRUD 代码）

ruoyi-framework：框架层与通用配置（安全、权限、拦截器、统一异常、公共基类等）
